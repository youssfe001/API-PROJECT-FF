API Reference

            
Complete endpoint documentation for the Free Fire API proxy.

          


          


            
            

              

                GET
                /api/v1/account
              

              
Fetch full player profile including rank, clan, pet, cosmetics, and social info.

              

                
ParamTypeDescription

                
                  
regionstringServer region (IND, ME, SG, BR, US, ID, TW, TH, VN, PK, BD, CIS, RU)

                  
uidstringPlayer UID

                
              

              
curl "http://localhost:3000/api/v1/account?region=IND&uid=1633864660"

              
            


            
            

              

                GET
                /api/v1/playerstats
              

              
Battle Royale stats (solo/duo/squad) for the given player.

              

                
ParamTypeDescription

                
                  
regionstringServer region

                  
uidstringPlayer UID

                  
matchmodestringCAREER, RANKED, or NORMAL

                
              

              
curl "http://localhost:3000/api/v1/playerstats?region=IND&uid=1633864660&matchmode=CAREER"

              
            


            
            

              

                GET
                /api/v1/playertcstats
              

              
Clash Squad stats with combat records, streaks, and accuracy data.

              

                
ParamTypeDescription

                
                  
regionstringServer region

                  
uidstringPlayer UID

                  
matchmodestringCAREER, RANKED, or NORMAL

                
              

              
curl "http://localhost:3000/api/v1/playertcstats?region=IND&uid=1633864660&matchmode=CAREER"

              
            


            
            

              

                GET
                /api/v1/search
              

              
Fuzzy search for players by nickname keyword.

              

                
ParamTypeDescription

                
                  
regionstringServer region

                  
keywordstringNickname search term

                
              

              
curl "http://localhost:3000/api/v1/search?region=IND&keyword=DARK"

              
            


            
            

              

                GET
                /api/v1/bancheck
              

              
Check if a player is banned. Cross-references multiple upstream sources.

              

                
ParamTypeDescription

                
                  
uidstringPlayer UID

                
              

              
curl "http://localhost:3000/api/v1/bancheck?uid=1633864660"

              
            


            
            

              

                GET
                /api/v1/wishlist
              

              
Get a player's wishlist items with resolved item metadata and icons.

              

                
ParamTypeDescription

                
                  
regionstringServer region

                  
uidstringPlayer UID

                
              

              
curl "http://localhost:3000/api/v1/wishlist?region=IND&uid=1633864660"

              
            


            
            

              

                GET
                /api/items
              

              
Look up a single item by ID with full metadata.

              

                
ParamTypeDescription

                
                  
idstringItem ID

                
              

              
curl "http://localhost:3000/api/items?id=101000005"

              
            


            
            

              

                GET
                /api/items/search
              

              
Search the item catalog by name with pagination support.

              

                
ParamTypeDescription

                
                  
qstringSearch query

                  
limitnumberMax results (default 20)

                
              

              
curl "http://localhost:3000/api/items/search?q=AK47&limit=10"

              
            


            
            

              

                GET
                /api/health
              

              
Server health check with uptime and status.

              
curl "http://localhost:3000/api/health"

              
            


            
            

              

                GET
                /api/guest-generator
              

              
Generate guest credentials for a given region.

              

                
ParamTypeDescription

                
                  
regionstringServer region

                
              

              
curl "http://localhost:3000/api/guest-generator?region=IND"

              
            


            
            

              

                GET
                /api/uid-generator
              

              
Generate random encrypted UIDs for testing.

              

                
ParamTypeDescription

                
                  
modestringGeneration mode (random)

                  
countnumberHow many UIDs to generate

                  
lengthnumberUID digit length

                
              

              
curl "http://localhost:3000/api/uid-generator?mode=random&count=5&length=10&teamXdarks=1"

              
            


            
            

              

                POST
                /api/like-spam
              

              
Send multiple like requests to a player. Requires region, uid, count, and delayMs in JSON body.

              

                
Body KeyTypeDescription

                
                  
regionstringServer region

                  
uidstringTarget player UID

                  
countnumberNumber of likes (1-500)

                  
delayMsnumberDelay between requests in ms (0-5000)

                
              

              
curl -X POST "http://localhost:3000/api/like-spam" -H "Content-Type: application/json" -d '{"region":"IND","uid":"1633864660","count":10,"delayMs":100}'
